# Sentiment Analysis using RNN

## Overview
This project performs sentiment analysis on text data using a Recurrent Neural Network (RNN). The model classifies input text into sentiment categories such as positive or negative based on learned word patterns and context.

---

## Features
- Text preprocessing and tokenization
- Word embedding representation
- RNN-based deep learning model for sequence learning
- Sentiment classification (Positive / Negative)
- Trained using labeled text dataset

---

## Tech Stack
- Python
- TensorFlow / Keras
- NumPy
- Pandas
- NLP (Natural Language Processing)

---

## Model Architecture
- Embedding Layer (converts words into dense vectors)
- RNN Layer (captures sequential patterns in text)
- Dense Layer (final classification)
- Output Layer (sigmoid/softmax for sentiment prediction)

---

## Workflow
1. Load and clean dataset
2. Tokenize text and convert to sequences
3. Apply padding to ensure uniform input size
4. Build RNN model with embedding layer
5. Train model on labeled sentiment dataset
6. Evaluate performance on test data

---

## Applications
- Movie review analysis
- Social media sentiment tracking
- Customer feedback analysis
- Opinion mining systems

---

## Future Improvements
- Replace RNN with LSTM/GRU for better performance
- Use pretrained embeddings (GloVe / Word2Vec)
- Deploy as web application using Flask or FastAPI
- Extend to multi-class sentiment classification
